Financial Analysis
Project Overview

This project focuses on analyzing financial data and creating an interactive dashboard to track revenue, expenses, profit, and budget performance.

The dataset was cleaned using Python, analyzed using SQL, and visualized in Power BI.
----

Tools Used
 - Python (Pandas)
 - SQL (SQLite)
 - Power BI
-----

Data Cleaning
The following data cleaning steps were performed:
 - Checked missing values
 - Removed duplicate records
 - Verified data types
 - Created a cleaned dataset for analysis
----

Key Performance Indicators (KPIs)
 - Total Revenue
 - Total Expenses
 - Net Profit
 - Profit Margin
 - Budget Variance
----

SQL Analysis
The following analyses were performed:
 - Monthly/Quarterly Revenue Trend
 - Department-wise Expenses
 - Region-wise Profit
 - Budget vs Actual Spending
-----

Dashboard Features
 - KPI Cards
 - Revenue Trend Analysis
 - Department Expense Analysis
 - Profit Breakdown
 - Budget Variance Monitoring
 - Region and Department Filters
----
